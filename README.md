# Molten Syntax

A volcanic syntax theme for Atom. Based on Simurai's [DuoTone Syntax]().

![screenshot]()

- C

- HTML
- Java
- JavaScript
- JSON
- Less
- Perl
- PHP
- Python
- Ruby
- SASS
- SCSS
- Stylus
- (La-)TeX
- XML
- YAML
